# Markdown Hub - The Swiss Army Knife for Document Conversion

Transform your documents effortlessly with Markdown Hub, the ultimate VSCode extension for seamless file format conversion. Whether you're working with Markdown, Office documents, or diagrams, this extension has you covered.

## 🚀 Key Features

### Markdown Conversion
- **Markdown → DOCX**: Convert `.md` files to professionally formatted Word documents with custom templates
- **Markdown → PDF**: Generate clean, print-ready PDF documents from your Markdown files
- **Markdown → HTML**: Create styled web pages with embedded CSS from Markdown content
- **Markdown → PPTX**: Transform Markdown into PowerPoint presentations automatically
- **SVG Support**: Automatically converts SVG code blocks in Markdown to PNG images in output documents

### Office & Document Conversion
- **DOCX → Markdown**: Extract content from Word documents into clean Markdown format
- **XLSX → Markdown**: Convert Excel spreadsheets to structured Markdown tables
- **PDF → Markdown**: Extract text and structure from PDF files (with OCR support for scanned documents)
- **SVG/Draw.io → PNG**: Convert vector graphics and flowcharts to high-quality PNG images

### Batch Processing
- **Bulk Markdown Conversion**: Process entire directories of `.md` files to DOCX, PDF, HTML, or PPTX
- **Bulk Office Conversion**: Convert multiple PDF, DOCX, PPTX, or Excel files to Markdown simultaneously
- **Smart Directory Processing**: Automatically handles nested folder structures

## 💡 Why Choose Markdown Hub?

- **One-Click Conversion**: Right-click any supported file or folder in VS Code Explorer
- **Template Support**: Use custom DOCX and PPTX templates for branded documents
- **Cross-Platform**: Works on Windows, macOS, and Linux
- **Batch Processing**: Handle multiple files efficiently
- **OCR Integration**: Extract text from scanned PDFs using Tesseract
- **Flexible Output**: Configurable output directories and file naming
- **Professional Results**: High-quality conversions suitable for business use

## 🛠️ Quick Start

1. Install the extension from VS Code Marketplace
2. Install Python dependencies: `pip install -r backend/requirements.txt`
3. Right-click any supported file in VS Code Explorer
4. Select your desired conversion option
5. Find converted files in your configured output directory

## ⚙️ System Requirements

### All Platforms
- Python 3.8 or higher
- Pandoc (for document conversion)

### Windows
- Microsoft Word (for DOCX conversion)
- Tesseract OCR (for PDF text recognition)

### macOS/Linux
- LibreOffice (for DOCX conversion)
- Tesseract OCR (for PDF text recognition)

### Optional Dependencies
- Cairo graphics library (for advanced SVG conversion)
- Inkscape (alternative SVG converter)
